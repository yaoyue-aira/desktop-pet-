# 🐾 桌面小宠物 Desktop Pet

一个可以更换头像和身体的桌面宠物，支持上传任意照片作为头像——你自己的、爱豆的、喜欢的人的，都可以！

工作累了需要分泌点多巴胺和肾上腺素？放上喜欢的人的头像做桌宠，瞬间斗志满满，继续搬砖！

如果你要说"真是牛马腌入味儿了，做个桌宠都是为工作服务的"——哎你怎么这么聪明，身体是可以换成牛和马的，快来试试吧~

## ✨ 功能
- 🐄 多种身体可选：Q版西装、西装、Q版裙子、裙子、奶牛、马
- 📷 上传照片自动识别人脸，裁剪成圆形头像
- 🚶 宠物会自己在屏幕上走来走去
- 🖱️ 可以用鼠标拖着它到处跑
- 💬 偶尔会冒出对话泡泡

## 📥 下载使用
1. 点击进入 `dist` 文件夹
2. 下载 `pet.exe`
3. 双击运行，无需安装任何东西！

## 🎮 操作方式
- **左键拖拽**：拖着宠物移动
- **右键**：打开菜单，上传头像 / 更换身体 / 退出

## 💻 系统要求
- Windows 10 / 11
